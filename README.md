# منصة اللوجستيات — Logistics Platform

نظام تشغيلي يومي لإدارة العمليات اللوجستية في المنطقة الجنوبية.

## التثبيت والتشغيل

```bash
# 1. فك الضغط عن المجلد
cd logistics-platform

# 2. تثبيت المكتبات
npm install

# 3. تشغيل بيئة التطوير
npm run dev
# يفتح على http://localhost:5173

# 4. بناء نسخة الإنتاج
npm run build
```

## الوحدات المبنية

| الوحدة | الوصف |
|--------|-------|
| 📊 لوحة التحكم | نظرة عامة على العمليات اليومية |
| 📦 إدارة الطلبات | استعراض، بحث، فلترة، تصدير |
| 🚛 إدارة الرحلات | إنشاء رحلات، متابعة الحالة |
| 📅 التخطيط الأسبوعي | خوارزمية توزيع تلقائي للطلبات |
| 🚚 إدارة الأسطول | بيانات الشاحنات والأبعاد |
| 📐 محاكاة التحميل | 2D bin packing visualization |
| ⏱ الوقت الإضافي | حساب رحلة + داخلي + تقارير شهرية |
| 👥 الموظفون | السائقون والمنسقون |
| 🔧 الصيانة | سجل الأعطال والتكاليف |
| 📈 مؤشرات الأداء | KPI بالسائق والشاحنة |
| 📋 التقارير | تصدير Excel، طباعة |
| 📥 الاستيراد | Excel من Dynamics 365 |
| ⚙️ الإعدادات | كل القيم قابلة للتخصيص |

## المتطلبات

- Node.js 18+
- npm أو yarn

## البيانات

- كل البيانات تُحفظ في localStorage (لا سيرفر مطلوب)
- استيراد Excel من Dynamics 365 مباشرة
